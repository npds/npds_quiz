# npds_quiz

<img src="https://codeberg.org/npds/npds_quiz/raw/branch/main/npds_quiz/npds_quiz.png" alt="Image du module npds_quiz" width="100" height="100" />
Le module **npds_quiz** pour NPDS est compatible avec les versions NPDS **16 et supérieures** .  
Des Quiz pour votre communauté !

---

## Installation

### 1. Télécharger le module

- [Dernière version stable (LTR) (recommandée)](https://codeberg.org/npds/npds_quiz/archive/latest.zip)  
- [Version en cours de développement](https://codeberg.org/npds/npds_quiz/archive/main.zip)

Téléchargez l’archive ZIP de la version choisie.

---

### 2. Copier le dossier du module

Depuis l’archive ZIP téléchargée, **copiez uniquement le premier sous-dossier** contenu dans l’archive (qui correspond au dossier du module `npds_quiz`) dans le répertoire suivant de votre installation NPDS :  
`/modules`



> ⚠️ **Important :** Ne copiez pas le contenu entier de l’archive directement dans `/modules`, mais uniquement le dossier racine extrait de l’archive.

---

### 3. Installer via l’interface d’administration

1. Connectez-vous en tant qu’administrateur à votre NPDS.
2. Rendez-vous dans le menu :  `/admin.php?op=modules`
3. Dans le tableau des modules, trouvez la ligne correspondant au module **npds_quiz**.
4. Cliquez sur l’icône **Installer le module** pour activer le module.

---

### Remarques

- La version stable (LTR) est recommandée pour un environnement de production, garantissant stabilité et support à long terme.  
- La version de développement contient les dernières fonctionnalités mais peut être moins stable.

---

N’hésitez pas à consulter la documentation complète du module pour plus d’informations et d’options de configuration.
## Désinstallation
Pour désinstaller le module utiliser uniquement l'interface administration !
1. Connectez-vous en tant qu’administrateur à votre NPDS.
2. Rendez-vous dans le menu :  `/admin.php?op=modules`
3. Dans le tableau des modules, trouvez la ligne correspondant au module **npds_quiz**.
4. Cliquez sur l’icône **Désinstaller le module** pour désactiver le module.
